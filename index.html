Welcome to my GitHub page :>

Here you will be able to find the download and documentation for my tool, ASearch.

ASearch is a privacy-focused web-scraping tool that is completely controlled from the terminal. ASearch can be utilized
to download images, grab elements, search the web, and more.

ASearch does not collect user data and browser files are deleted after each session. ASearch does have a feature that locally
logs function calls and actions. However, this is disabled by default and is primarily intended for debugging purposes.

ASearch can be downloaded for both Windows and Linux. A list of requirements can be found in the 'requirements.txt' file
or below.

ASearch also has proxy and TOR support for the extra paranoid
