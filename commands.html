PENTESTING

Useful for vulnerability testing


findelem Find all elements in a website by type
--url TEXT URL to retrieve [required]
-fe, --findelem TEXT Find all of specified element on webpage [required]
--help Show this message and exit.

inject Find element using a variety of different identifiers Inject queries into given text element (if unsure run findelem first)
-by, --elemidentifier [ID|XPATH|CSS|name|CLASS_NAME|TAG_NAME] Specify how you want to identify an element [required]
--element TEXT element name [required]
--url TEXT URL to retrieve [required]
-input, --inject
--help Show this message and exit.

lookup Get WHOIS information on given URL
--url TEXT URL to retrieve [required]
--help Show this message and exit.

WEBSCRAPING

Useful for scraping information off the web

open Get contents of webpage (Body, Images, Urls. . . (experimental))
--url TEXT URL to retrieve [required]
-t, --text Get all text from given url
-img, --images Get all images from given url
-s, --scroll Scroll down to load more elements (takes longer)
-d, --download Download all images from URL, automatically creates a folder and places them inside
-ss, --screenshot Take screenshot of given URL
--help Show this message and exit.

send Send search query to google and duckduckgo
-q, --query TEXT Query to send (pass as STR) [required]
--help Show this message and exit.

dork Send dork query, can be randomly be generated from text files, to generate random dork pass "g"
-q, --query TEXT Query to send (pass as STR)
-p, --proxy TEXT Proxy to send request <IP:PORT>
--help Show this message and exit.
